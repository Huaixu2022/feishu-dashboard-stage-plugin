# 飞书多维表格仪表盘插件：阶段进度卡

## 功能
- 读取多维表格中“是否当日=是”的记录
- 自动显示当前阶段、当前工作日、本月总工作日、工作日进度
- 自动计算阶段范围：阶段1、阶段2、阶段3
- 可选显示阶段目标、当前完成，并给出达标/未达标提示

## 必备字段
表里需要有这些字段：
- 是否当日：今天这一行为“是”
- 本月总工作日数：例如 21
- 当月第几个工作日：例如 11
- 当前日期所处工作日进度：例如 52.38% 或 0.5238

可选字段：
- 当前日期所处阶段：第一阶段/第二阶段/第三阶段
- 阶段目标进度
- 实际完成进度

## 本地运行
```bash
npm install
npm run dev
```

## 打包
```bash
npm run build
```

## 飞书使用
1. 将项目部署到 Vercel、Github Pages 或自己的服务器
2. 打开多维表格仪表盘
3. 添加组件 → 更多 → 添加自定义插件
4. 填入运行地址
5. 进入配置页选择对应字段

注意：vite.config.js 中必须保留 `base: './'`。
