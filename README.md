# 飞书多维表格仪表盘插件：阶段进度卡

自动读取多维表格中「是否当日 = 是」的记录，并展示：当前阶段、当前工作日、工作日进度、三个阶段范围。

## 需要的字段名

插件默认读取以下字段，字段名必须一致：

- 是否当日
- 本月总工作日数
- 当月第几个工作日
- 当前日期所处阶段
- 当前日期所处工作日进度
- 当前阶段编码
- 第一阶段截止
- 第二阶段截止

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

上传到 GitHub 后，用 Vercel 导入仓库部署。部署成功后，把 Vercel 地址填入飞书仪表盘「添加自定义插件」。
